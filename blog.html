What is the Document Object Model (DOM)?

The Document object Model (DOM) is a programing interface for HTML and XML Document. which contain Html elements for manipulate by Javascript. It's as a tree of object

  • QuerySelector()
  • QuerySelectorAll()

    • What is event delegation in the context of the DOM, and why is it useful?

    Event Delegation is a technique used in the DOM to handle on events on ancestor element as an alternative of attaching event listeners to sole child element.This proceed towards takes advantage of the fact that event bubble up the DOM tree.
    Event delegation is useful for many reasons:

  • It reduces the number of event listeners attached to the DOM
  • Attaching a individual event listener top ancestor element is more qualityful than attaching multiple event listener to individual child element
  • Event Delegation makes it easy to handle events on a large number of child element

    • How do you manipulate an element's attributes and styles using the DOM?

    The Dom provides many ways to manipulate an element's atributes and style.
    Some Commons method:

  • setAttribute() and removeAttribute() to add or remove attributes
  • style.cssText to add or remove style
  • style.propertyName to add or remove style
  • classList.add() and classList.remove() to add or remove class
  • classList.toggle() to add or remove class
  • style.display = "none" to hide an element
